# ORB-SLAM2 for Ardupilot

Forked from https://github.com/raulmur/ORB_SLAM2, the intention of this project is to modify ORB-SLAM2 to support integration with [Ardupilot](https://github.com/ArduPilot/ardupilot). Feel free to chat via gitter here: https://gitter.im/ArduPilot/ardupilot/VisionProjects


**Original Authors:** [Raul Mur-Artal](http://webdiis.unizar.es/~raulmur/), [Juan D. Tardos](http://webdiis.unizar.es/~jdtardos/), [J. M. M. Montiel](http://webdiis.unizar.es/~josemari/) and [Dorian Galvez-Lopez](http://doriangalvez.com/) ([DBoW2](https://github.com/dorian3d/DBoW2))
